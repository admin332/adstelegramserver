# Adsingo

Маркетплейс Telegram-рекламы с безопасными сделками (Escrow), верифицированной статистикой каналов и автопостингом.

## Технологии

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Backend)
- TON Blockchain (Escrow-платежи)

## Разработка

Разработано командой AGDX.

## Локальный запуск

```sh
npm install
npm run dev
```


# Изменение позиции стикера

## Текущее положение

Сейчас стикер позиционируется так:
```
-top-12 left-1/2 -translate-x-1/2
```
- Центрирован по горизонтали
- Смещён вверх на 3rem (48px)

## Новое положение

Для смещения **правее и выше**:
```
-top-16 left-1/2 translate-x-4
```
- `-top-16` — поднимаем выше (было -12, стало -16, т.е. на 1rem выше)
- `translate-x-4` — сдвигаем вправо на 1rem (16px) вместо центрирования

## Изменения

**Файл:** `src/pages/Index.tsx` (строка 183)

Заменить:
```tsx
<div className="absolute -top-12 left-1/2 -translate-x-1/2 z-10 pointer-events-none">
```

На:
```tsx
<div className="absolute -top-16 left-1/2 translate-x-4 z-10 pointer-events-none">
```

## Результат

Стикер будет отображаться немного правее центра и выше от первой карточки канала.

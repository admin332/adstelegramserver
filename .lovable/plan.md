

## Цель
Переместить таймер обратного отсчёта в правый верхний угол карточки сделки для лучшей видимости.

## Текущее расположение

Таймер сейчас находится в нижней части карточки рядом со статусом:
```
┌────────────────────────────────────┐
│ [Avatar] Название канала           │
│          @username                 │
│                                    │
│ 💎 10 TON ≈ $35 • 2 поста • 24ч    │
│────────────────────────────────────│
│ [Статус]              ⏱️ 15:42 • 5м │  ← таймер здесь
│ [Оплатить]                         │
└────────────────────────────────────┘
```

## Новое расположение

Таймер будет в правом верхнем углу:
```
┌────────────────────────────────────┐
│ [Avatar] Название канала   ⏱️ 15:42 │  ← таймер вверху справа
│          @username                 │
│                                    │
│ 💎 10 TON ≈ $35 • 2 поста • 24ч    │
│────────────────────────────────────│
│ [Статус]                       5м  │
│ [Оплатить]                         │
└────────────────────────────────────┘
```

## Изменения в файле

**`src/components/DealCard.tsx`**

1. Добавить `relative` к главному контейнеру карточки для позиционирования таймера

2. Переместить `ExpirationTimer` в верхнюю часть карточки с абсолютным позиционированием:
   - Добавить `absolute top-4 right-4` для размещения в правом верхнем углу
   - Обернуть в условный рендер только для `pending` статуса с `expiresAt`

3. Удалить таймер из нижней секции (строки 169-171)

## Технические детали

```tsx
// Главный контейнер - добавить relative
<div className="bg-card rounded-2xl p-4 animate-fade-in relative">

  {/* Таймер в правом верхнем углу */}
  {status === "pending" && expiresAt && (
    <div className="absolute top-4 right-4">
      <ExpirationTimer expiresAt={expiresAt} />
    </div>
  )}

  {/* Остальной контент без изменений */}
  ...
</div>
```

Нижняя секция со статусом останется без таймера:
```tsx
<div className="flex items-center justify-between mt-4 pt-3 border-t border-white/5">
  <div className={cn("flex items-center gap-1.5 px-2 py-1 rounded-full", config.bgColor)}>
    <StatusIcon className={cn("w-4 h-4", config.color)} />
    <span className={cn("text-sm font-medium", config.color)}>{config.label}</span>
  </div>
  <span className="text-sm text-muted-foreground">{timeAgo}</span>
</div>
```

## Результат

Таймер обратного отсчёта будет сразу виден в правом верхнем углу карточки, что сделает его более заметным для пользователя.

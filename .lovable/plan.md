
# Изменение формы создания кампании (Шаг 1)

## Текущий вид

```
Название кампании
Укажите название для вашей рекламной кампании

Название *                        ← УБРАТЬ
[Например: Продвижение приложения] ← ИЗМЕНИТЬ
```

## Новый вид

```
Название кампании
Укажите название для вашей рекламной кампании

[Например: Реклама сайта]
```

## Изменения

**Файл:** `src/components/create/CreateCampaignForm.tsx`

### 1. Удалить Label (строка 249)

```tsx
// Удалить эту строку:
<Label htmlFor="name">Название *</Label>
```

### 2. Изменить placeholder в Input (строка 252)

```tsx
// Было:
placeholder="Например: Продвижение приложения"

// Станет:
placeholder="Например: Реклама сайта"
```

## Результат

Форма станет более лаконичной — заголовок "Название кампании" уже понятно объясняет что нужно ввести, дополнительный лейбл избыточен.

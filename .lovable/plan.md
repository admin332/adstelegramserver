
# Центрирование изображения канала в карточке

## Текущее состояние

Сейчас изображение канала позиционируется справа в правой половине карточки:
- Контейнер обрезается через `clipPath: 'polygon(50% 0, 100% 0, 100% 100%, 50% 100%)'` — видна только правая половина карточки
- Изображение использует `object-contain object-right` — выравнивание по правому краю

## Проблема

Изображение прижато к правому краю, а не отцентровано внутри видимой области (правой половины карточки).

## Решение

Изменить выравнивание изображения с `object-right` на `object-center`, чтобы контент изображения был отцентрован внутри видимой области.

## Техническое изменение

**Файл:** `src/components/ChannelCard.tsx` (строка 108)

Заменить:
```tsx
className="w-full h-full object-contain object-right"
```

На:
```tsx
className="w-full h-full object-contain object-center"
```

## Результат

Изображение канала будет отображаться по центру видимого контейнера (правой половины карточки), что улучшит визуальное восприятие — главный объект изображения (например, логотип или персонаж) будет находиться в центре видимой области.

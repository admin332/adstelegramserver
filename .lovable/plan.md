

# Перемещение стикера за карточку канала

## Текущее состояние

Сейчас стикер позиционируется с `z-10`, что помещает его **поверх** карточки канала.

## Изменение

Для отображения стикера **сзади** карточки нужно уменьшить z-index:

**Файл:** `src/pages/Index.tsx` (строка 183)

Заменить:
```tsx
<div className="absolute -top-20 left-1/2 translate-x-6 z-10 pointer-events-none">
```

На:
```tsx
<div className="absolute -top-20 left-1/2 translate-x-6 z-0 pointer-events-none">
```

## Результат

Стикер будет отображаться за карточкой канала — видимая часть анимации будет только там, где она выходит за пределы карточки.

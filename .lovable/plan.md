

## Изменение лейблов цен в настройках канала

Простое изменение текста в секции цен на странице настроек канала.

---

## Что нужно изменить

| Сейчас | После |
|--------|-------|
| `24 часа` | `24 часа / 1 пост` |
| `48 часов` | `24 часа / 2+ поста` |

---

## Файл: `src/pages/ChannelSettings.tsx`

**Строка 190** — изменить текст первой цены:
```tsx
// Было:
<span className="text-muted-foreground">24 часа</span>

// Станет:
<span className="text-muted-foreground">24 часа / 1 пост</span>
```

**Строка 206** — изменить текст второй цены:
```tsx
// Было:
<span className="text-muted-foreground">48 часов</span>

// Станет:
<span className="text-muted-foreground">24 часа / 2+ поста</span>
```

---

## Результат

Рекламодатели сразу увидят связь между ценой и количеством постов — это соответствует логике выбора при оформлении заказа.

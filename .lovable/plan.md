### Онбординг заказа (модальное окно / drawer)

**Шаг 1: Выбор количества постов**

```
┌──────────────────────────────────────────┐
│                                          │
│  ═══════════●═══════════  Шаг 1 из 3    │
│                                          │
│          Выберите количество            │
│               постов                     │
│                                          │
│  ┌──────────────────────────────────────┐│
│  │                                      ││
│  │      [ - ]     2      [ + ]          ││
│  │                                      ││
│  │        постов на 24 часа             ││
│  │                                      ││
│  └──────────────────────────────────────┘│
│                                          │
│           Стоимость заказа               │
│                                          │
│  ┌──────────────────────────────────────┐│
│  │  1 пост × 50 TON = 100 TON           ││
│  └──────────────────────────────────────┘│
│                                          │
│       ┌──────────────────────┐           │
│       │       Далее →        │           │
│       └──────────────────────┘           │
│                                          │
└──────────────────────────────────────────┘
```

**Логика:**
- Начальное количество: 1 пост
- Кнопки `+` и `-` для изменения
- Минимум: 1, максимум: 10
- Сумма = количество × цена за пост
- Прогресс-бар показывает текущий шаг

### Технические детали

#### 1. Обновление mockChannels (добавление данных)

```typescript
export interface Channel {
  // ... существующие поля
  description?: string;      // Описание канала
  successfulAds?: number;    // Количество успешных реклам
}

// Добавить к каждому каналу:

  description: "Канал о криптовалютах и блокчейне. Ежедневные новости, аналитика рынка и обзоры проектов.",
  successfulAds: 156,
}
```

#### 2. Страница Channel.tsx

- Получение `id` канала из URL через `useParams()`
- Поиск канала в `mockChannels` по id
- Hero-секция с фоном и аватаркой
- Статистика в 3 колонки
- Блок описания
- Детальная статистика (рейтинг, вовлечённость, категория)
- Кнопка "Заказать рекламу" — открывает drawer

#### 3. OrderDrawer (Vaul drawer)

- Прогресс-бар сверху (1 из 3 шагов)
- Содержимое меняется по шагам
- На шаге 1: выбор количества постов
- Кнопка "Далее" (пока без перехода — стоп на шаге 1)

#### 4. PostQuantitySelector

- Кнопки `-` и `+` для изменения количества
- Отображение текущего количества
- Калькуляция итоговой суммы
- Анимация изменения цены

#### 5. Навигация с карточки

```typescript
// ChannelCard.tsx — кнопка "Купить"
<Button
  size="sm"
  onClick={(e) => {
    e.stopPropagation();
    navigate(`/channel/${id}`);
  }}
>
  Купить
</Button>
```

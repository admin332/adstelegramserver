

# Изображение на всю высоту карточки с обрезкой по ширине

## Проблема

Сейчас изображение использует `object-contain`, который уменьшает изображение, чтобы оно полностью поместилось в контейнер. Если изображение шире контейнера — оно уменьшается, и появляется пустое пространство сверху/снизу.

## Решение

Изменить `object-contain` на `object-cover`:
- Изображение всегда заполняет контейнер по высоте (100% высоты карточки)
- Если изображение шире контейнера — лишняя ширина обрезается
- `object-center` сохраняется — обрезка будет симметричной с обеих сторон

## Техническое изменение

**Файл:** `src/components/ChannelCard.tsx` (строка 103)

**Текущий код:**
```tsx
className="w-full h-full object-contain object-center"
```

**Новый код:**
```tsx
className="w-full h-full object-cover object-center"
```

## Разница между режимами

| Свойство | object-contain | object-cover |
|----------|---------------|--------------|
| Поведение | Уменьшает изображение, чтобы полностью поместилось | Заполняет контейнер, обрезая лишнее |
| Высота | Может быть меньше контейнера | Всегда 100% контейнера |
| Обрезка | Нет | Да, по ширине если изображение шире |

## Результат

Изображение канала всегда будет занимать полную высоту карточки (192px / h-48), а если изображение слишком широкое — оно будет обрезано по краям, сохраняя центр в видимой области.

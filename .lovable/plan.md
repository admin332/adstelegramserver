

## Изменение минимального значения времени до публикации

Изменю минимальное значение слайдера с 0 на 2 часа, чтобы соответствовать логике покупки рекламы.

---

## Что нужно изменить

| Параметр | Сейчас | После |
|----------|--------|-------|
| Минимум слайдера (`min`) | `0` | `2` |
| Метки под слайдером | `[0, 6, 12, 24, 48, 72]` | `[2, 6, 12, 24, 48, 72]` |
| Отображаемое значение | `+{value \|\| 0}ч` | `+{value \|\| 2}ч` |
| Дефолтное значение | `0` | `2` |

---

## Файл: `src/pages/ChannelSettings.tsx`

**Строка 263** — изменить дефолтное отображение:
```tsx
// Было:
+{localSettings.min_hours_before_post || 0}ч

// Станет:
+{localSettings.min_hours_before_post || 2}ч
```

**Строка 268** — изменить дефолтное значение слайдера:
```tsx
// Было:
value={[localSettings.min_hours_before_post || 0]}

// Станет:
value={[localSettings.min_hours_before_post || 2]}
```

**Строка 271** — изменить минимум слайдера:
```tsx
// Было:
min={0}

// Станет:
min={2}
```

**Строка 277** — изменить метки под слайдером:
```tsx
// Было:
{[0, 6, 12, 24, 48, 72].map((h) => (

// Станет:
{[2, 6, 12, 24, 48, 72].map((h) => (
```

---

## Результат

Минимальное время до публикации теперь начинается с 2 часов — это соответствует ограничению при покупке рекламы, где уже установлен минимальный буфер в 2 часа.


## Цель
Переместить отображение "Команда канала" со страницы просмотра канала (`/channel/:id`) на страницу "Мои каналы" в разделе управления.

## Изменения

### 1. Убрать ChannelTeam со страницы Channel.tsx

**Файл**: `src/pages/Channel.tsx`

- Удалить импорт `ChannelTeam` (строка 10)
- Удалить рендер компонента `<ChannelTeam channelId={id} />` (строка 156)

### 2. Добавить отображение команды в MyChannelsList.tsx

**Файл**: `src/components/create/MyChannelsList.tsx`

- Импортировать `useChannelAdmins` и необходимые компоненты (`Avatar`, `Crown`, `Shield`)
- Добавить раскрывающийся блок команды под каждой карточкой канала
- Показывать список админов канала в компактном виде:
  - Аватар пользователя
  - Имя
  - Роль (Владелец/Менеджер) с иконкой

### Примерный UI под карточкой канала

```
┌─────────────────────────────────────────┐
│ [Avatar] Channel Name  ✓  [Менеджер]   │
│          @username                      │
│          👥 10K · Криптовалюты          │
├─────────────────────────────────────────┤
│ Команда:                                │
│  👤 Иван Иванов (👑 Владелец)           │
│  👤 Петр Петров (🛡️ Менеджер)          │
└─────────────────────────────────────────┘
```

## Технические детали

### Компонент ChannelTeamCompact (новый)
Создать облегченную версию для списка каналов:
- Без анимаций motion
- Компактный стиль (меньше padding)
- Показывать только имя и роль (без полных permissions)

Или можно встроить напрямую в `MyChannelsList`, делая запрос `useChannelAdmins(channel.id)` для каждого канала.

### Оптимизация
- Запросы к `channel_admins` происходят параллельно для всех каналов пользователя
- Используем React Query кэширование

## Результат
- Страница канала для покупателей - без лишней информации о команде
- Страница "Мои каналы" - владелец/менеджер видит всю команду под каждым каналом
